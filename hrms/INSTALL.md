# Установка и запуск HRMS System

## Требования

- Windows 10/11
- Python 3.10 или выше
- Microsoft Excel (для xlwings)
- Git (опционально)

## Установка

### 1. Установка Python

1. Скачайте Python 3.10+ с [python.org](https://www.python.org/downloads/)
2. При установке **обязательно** поставьте галочку "Add Python to PATH"
3. Проверьте установку:
   ```bash
   python --version
   ```

### 2. Установка зависимостей

Запустите `setup.bat` - он автоматически:
- Создаст виртуальное окружение `.venv`
- Установит все необходимые библиотеки
- Настроит проект

```bash
setup.bat
```

### 3. Настройка Excel

1. Откройте файл `otdel-kadrov.xlsm`
2. Разрешите выполнение макросов (если запросит)
3. Убедитесь что есть листы:
   - Сотрудники
   - Отпуска
   - Настройки
   - Журнал событий

## Тестирование

### Тест базовой инфраструктуры

1. Откройте `otdel-kadrov.xlsm` в Excel
2. Нажмите `Alt+F11` для открытия редактора VBA
3. Найдите модуль с макросами
4. Запустите макрос `TestInfrastructure`

Или добавьте кнопку на лист:
1. Вкладка "Разработчик" → "Вставить" → "Кнопка"
2. Назначьте макрос `TestInfrastructure`
3. Нажмите кнопку

Тест проверит:
- ✓ Подключение к Excel
- ✓ Чтение сотрудников
- ✓ Чтение справочников
- ✓ Чтение отпусков

## Troubleshooting

### Python не найден
- Переустановите Python с галочкой "Add to PATH"
- Перезагрузите компьютер

### xlwings не работает
- Убедитесь что Excel установлен
- Запустите: `pip install xlwings`
- Проверьте: `python -c "import xlwings; print('OK')"`

### Ошибка подключения к Excel
- Запускайте скрипты ТОЛЬКО из Excel через VBA
- Не запускайте напрямую через `python test_infrastructure.py`

### Логи не создаются
- Проверьте что папка `logs/` существует
- Проверьте права доступа к папке
