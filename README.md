# HRMS System - Система управления персоналом

Полнофункциональная HRMS (Human Resource Management System) на базе Python и Excel.

## Описание

Система автоматизирует работу отдела кадров:
- Контроль окончания контрактов с алертами
- Отслеживание дней рождения сотрудников
- Управление отпусками с проверкой пересечений
- Генерация приказов из Word-шаблонов
- Управление личными делами сотрудников
- Архивирование уволенных сотрудников

## Архитектура

- **Excel** = база данных (чистые таблицы)
- **Python** = обработчик данных и бизнес-логика
- **Запуск**: ТОЛЬКО из Excel через VBA макрос

## Технологии

- Python 3.10+
- xlwings - интеграция с Excel
- pandas - обработка данных
- customtkinter - современный GUI
- docxtpl - генерация Word документов
- loguru - логирование

## Установка

1. Убедитесь что установлен Python 3.10+ (с галочкой "Add Python to PATH")
2. Запустите `setup.bat` для автоматической установки зависимостей
3. Откройте Excel файл `otdel-kadrov.xlsm`
4. Нажмите кнопку запуска HRMS в Excel

## Структура проекта

```
hrms-system/
├── main.py                 # Точка входа приложения
├── settings.py             # Конфигурация
├── requirements.txt        # Зависимости Python
├── macro.vba              # VBA макрос для запуска из Excel

├── .kiro/specs/           # Спецификации и документация
└── otdel-kadrov.xlsm     # Excel база данных (НЕ в Git!)
```

## Документация

Полная документация находится в `.kiro/specs/hrms-system/`:
- `requirements.md` - требования к системе
- `design.md` - технический дизайн
- `tasks.md` - план реализации
- `DESIGN_DECISIONS.md` - ключевые решения

## Безопасность

⚠️ **ВАЖНО**: 
- Excel файл с данными сотрудников НЕ коммитится в Git
- Папки `личные_дела/` и `приказы/` исключены из Git
- Все конфиденциальные данные хранятся локально

## Разработка

Следуйте плану реализации в `tasks.md`:
- Фаза 1-3: Инфраструктура и аналитика
- Фаза 4-6: GUI компоненты
- Фаза 7-9: Генерация приказов и операции с сотрудниками
- Фаза 10-12: Личные дела, обработка ошибок, интеграция
