======================================================
 ИНСТРУКЦИЯ: КАК ЗАПУСКАТЬ PYTHON-СКРИПТЫ ИЗ EXCEL
======================================================

ШАГ 1: АВТОМАТИЧЕСКАЯ УСТАНОВКА
1. Убедитесь, что у вас установлен Python (при установке обязательно 
   ставьте галочку "Add Python to PATH" в самом низу окна!).
2. Закройте все окна Excel.
3. Запустите файл `setup.bat`. Он сам скачает нужные библиотеки и 
   добавит надстройку xlwings в ваш Excel.

ШАГ 2: ПОДГОТОВКА EXCEL
1. Откройте ваш файл (например, otdel-kadrov.xlsm).
2. Убедитесь, что в верхнем меню Excel появилась вкладка "xlwings".

ШАГ 3: НАСТРОЙКА VBA (ЧТОБЫ НЕ БЫЛО ОШИБОК)
1. Нажмите Alt + F11 (откроется редактор VBA).
2. В самом верхнем меню нажмите: Tools (Сервис) -> References... (Ссылки...).
3. В открывшемся длинном списке прокрутите вниз, найдите пункт "xlwings" 
   и ПОСТАВЬТЕ ВОЗЛЕ НЕГО ГАЛОЧКУ [v]. Нажмите OK.
   (Если этого не сделать, Excel выдаст ошибку "Compile error: Sub or Function not defined").

ШАГ 4: ДОБАВЛЕНИЕ МАКРОСА
1. В том же редакторе VBA слева нажмите правой кнопкой мыши по вашему файлу:
   Insert -> Module.
2. В большое белое окно справа вставьте следующий код:

Sub RunMyPython()
    RunPython "import ваш_скрипт; ваш_скрипт.ваша_функция()"
End Sub

(Например: RunPython "import sort_employees; sort_employees.show_employee_selector()")

ШАГ 5: ЗАПУСК
1. Закройте редактор VBA.
2. В самом Excel вы можете нарисовать любую фигуру или кнопку (Вставка -> Фигуры).
3. Кликните по ней правой кнопкой мыши -> "Назначить макрос".
4. Выберите `RunMyPython` и нажмите OK.
5. Готово! Теперь по клику на кнопку будет запускаться ваш Python-код.

ВАЖНО: Ваш файл скрипта (.py) должен лежать в той же папке, что и файл Excel!
